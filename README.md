# Scrabble

Give a variable number of words, a program to output the word with the maximum score, based on the following scoring system:

1 point = a,e,i,l,n,o,r,s,t,u

2 points = d,g

3 points = b,c,m,p

4 points = f,h,v,w,y

5 points = k

8 points = j,k

10 points = q,z

Max word length = 10 letters
